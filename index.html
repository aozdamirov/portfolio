À quoi sert une veille technologique ?

La veille technologique consiste à surveiller les évolutions des technologies pour anticiper les changements, identifier les opportunités d'innovation et rester compétitif. Elle permet de suivre les nouvelles tendances, d'analyser les innovations dans le secteur et d'adapter les stratégies en fonction des évolutions technologiques.


Comment faire une veille technologique ?

Plusieurs méthode existe pour faire une veille technologique : push et pull.


Méthode push :

Alertes par mail, abonnement à des newsletters, flux RSS, applications mobile (Feedly), extensions (daily.dev)

Avantages : Gain de temps, car les informations arrivent automatiquement.


Inconvénients : Les informations peuvent être trop générales, parfois noyées sous un trop grand volume de données.


Méthode pull :

Recherche sur des sites spécialisés, exploration de blogs, forums ou publications scientifiques, réseaux sociaux.

Avantages : Permet d'obtenir des informations très ciblées et précises, adaptées aux besoins spécifiques.


Inconvénients : Cela demande plus de temps et d'effort, car l'utilisateur doit constamment rechercher et filtrer les informations pertinentes.


Cursor AI

Cursor AI est un editeur de code assisté par l'intelligence artificielle.

Cursor AI possède 3 modes : Chat, Composer et Agent.

1.Chat

  • - Guide de débugage en temps réel
  • - Réponses rapides aux questions de code

Exemple : faire une requete pour une partie spécifique du code et il va l'appliquer directement dans le fichier.

2.Composer

Generation de code pour les plus grandes taches, ce mode est capable de modifier plusieurs fichier avec plus d'autonomie que le mode Chat

Exemple : Composer est capable de comprendre la structure de code et peut générer plusieurs fichiers avec une seul requete.

3.Agent

Mode compltement autonome qui fonctionne sur l'entiéreté du projet. Ce mode genere du code dans plusieurs fichiers, execute des commandes shell, et réfléchis lui meme.

Exemple : Composer est capable de comprendre la structure de code et peut générer plusieurs fichiers avec une seul requete.
